What is Javascript?

Wikipedia defines JavaScript as a high-level, often just-in-time compiled, and multi-paradigm language

You will find JavaScript everywhere on the internet. JavaScript is used by web developers to make the web page more intuitive. JavaScript makes single page feel like an enterprize web application. JavaScript is also used on the server side to render webpages.

JavaScript allows developers (You😉) to create a interactive web pages. Make the web pages work for your user in a way that give your user a pleaseant experience.
